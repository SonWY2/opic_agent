# Roleplay - Work Demo Similar Experience

## 문제

- 한국어: 업무 중 시스템 문제를 해결했던 비슷한 경험을 말해 주세요.
- English: Talk about a similar experience when you solved a system issue at work.

## 암기 포인트 및 흐름

- 프로젝트 상황 -> 문제 발생 -> 대안 제시 -> 결과 -> 배운 점
- legacy migration, system improvement 표현을 자연스럽게 재사용한다.
- Q21의 "녹화 영상, 핵심 기능, 다시 진행" 흐름과 연결한다.

## 한국어 전문

네, 업무에서 비슷한 경험이 있습니다.

예전에 레거시 시스템을 새 시스템으로 옮기는 프로젝트에서 고객 데모를 준비한 적이 있습니다.

그런데 미팅 직전에 테스트 서버가 느려져서 일부 화면이 정상적으로 열리지 않았습니다.

저는 먼저 고객에게 상황을 설명하고, 불편을 드려 죄송하다고 말했습니다.

그리고 미리 준비해 둔 스크린샷과 짧은 영상을 사용해서 핵심 흐름을 설명했습니다.

그 후 서버가 안정된 다음 날에 실제 데모를 다시 진행했습니다.

다행히 고객은 상황을 이해해 주었고, 프로젝트 일정에도 큰 문제는 없었습니다.

그 경험 이후로 저는 중요한 데모 전에는 항상 백업 자료를 준비합니다.

문제가 생겨도 대안이 있으면 훨씬 차분하게 대응할 수 있습니다.

## 영어 전문

Yes, I have a similar work experience.

In the past, I prepared a client demo for a project that moved a legacy system to a new system.

But right before the meeting, the test server became slow, and some screens did not open properly.

I first explained the situation to the client and apologized for the inconvenience.

Then I used prepared screenshots and a short video to explain the key flow.

After the server became stable, we ran the actual demo again the next day.

Luckily, the client understood the situation, and there was no major issue with the project schedule.

Since then, I always prepare backup materials before an important demo.

When I have an alternative, I can handle problems much more calmly.

## 키 문장 및 단어

### Key Sentences

- The test server became slow.
- Some screens did not open properly.
- I explained the situation and apologized.
- I used backup materials.
- I can handle problems more calmly.

### Key Words

- test server : 테스트 서버
- properly : 제대로
- backup materials : 백업 자료
- calmly : 차분하게

## 키 한국어 문장 및 대응되는 영어표현 여러개

### 테스트 서버가 느려졌습니다.

- The test server became slow.
- The test server started responding slowly.
- The test environment was not stable.

### 일부 화면이 정상적으로 열리지 않았습니다.

- Some screens did not open properly.
- A few pages were not loading correctly.
- Some parts of the system did not work as expected.

### 백업 자료로 핵심 흐름을 설명했습니다.

- I used backup materials to explain the key flow.
- I walked them through the main process with screenshots.
- I explained the important parts using a short video.

### 대안이 있으면 차분하게 대응할 수 있습니다.

- When I have an alternative, I can handle problems more calmly.
- A backup plan helps me stay calm.
- It is easier to respond when I have another option.
