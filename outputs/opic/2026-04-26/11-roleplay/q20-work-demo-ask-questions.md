# Roleplay - Work Demo Ask Questions

## 문제

- 한국어: 고객사와 소프트웨어 데모 미팅을 잡기 전에 필요한 정보를 질문해 주세요.
- English: Before setting up a software demo meeting with a client, ask for the necessary information.

## 암기 포인트 및 흐름

- 미팅 목적 -> 참석자 -> 시간 -> 데모 환경 -> 자료 요청
- IT 직무 페르소나를 직접 활용한다.
- 질문은 업무 전화처럼 짧고 명확하게 한다.

## 한국어 전문

안녕하세요. 다음 주 소프트웨어 데모 미팅과 관련해서 몇 가지 확인하고 싶습니다.

먼저 미팅에 몇 분 정도 참석하실 예정인가요?

그리고 가장 편한 날짜와 시간이 언제인지 알려 주실 수 있을까요?

데모에서는 기존 시스템에서 새 시스템으로 데이터를 옮기는 흐름을 보여 드리려고 합니다.

혹시 특별히 보고 싶은 기능이나 화면이 있으신가요?

또한 화면 공유가 가능한 회의 도구를 사용하면 좋을 것 같습니다.

필요하다면 제가 데모 전에 짧은 안내 자료를 보내 드릴 수 있습니다.

확인해 주시면 일정과 자료를 맞춰 준비하겠습니다.

감사합니다.

## 영어 전문

Hello. I would like to check a few things about next week's software demo meeting.

First, how many people will join the meeting?

Also, could you let me know the date and time that works best for you?

In the demo, I plan to show how data moves from the legacy system to the new system.

Are there any specific features or screens you would like to see?

Also, it would be helpful to use a meeting tool that allows screen sharing.

If needed, I can send a short guide before the demo.

Once you confirm, I will prepare the schedule and materials accordingly.

Thank you.

## 키 문장 및 단어

### Key Sentences

- I would like to check a few things.
- How many people will join?
- What date and time works best for you?
- Are there any specific features you would like to see?
- I can send a short guide before the demo.

### Key Words

- software demo : 소프트웨어 데모
- legacy system : 레거시 시스템
- screen sharing : 화면 공유
- materials : 자료

## 키 한국어 문장 및 대응되는 영어표현 여러개

### 몇 분이 참석하실 예정인가요?

- How many people will join the meeting?
- How many attendees should I expect?
- Could you tell me how many people will be there?

### 가장 편한 날짜와 시간이 언제인가요?

- What date and time works best for you?
- Could you let me know your preferred time?
- When would be most convenient for you?

### 보고 싶은 기능이 있으신가요?

- Are there any specific features you would like to see?
- Is there any function you want me to focus on?
- Do you have any screens you want to check during the demo?

### 데모 전에 자료를 보내 드릴 수 있습니다.

- I can send a short guide before the demo.
- I can share the materials in advance.
- I can send a brief document before the meeting.
