# Roleplay - Cafe Ask Questions

## 문제

- 한국어: 카페에 가기 전에 직원에게 자리, 와이파이, 콘센트에 대해 질문해 주세요.
- English: You are calling a cafe. Ask about seats, Wi-Fi, and outlets.

## 암기 포인트 및 흐름

- 인사 -> 방문 목적 -> 자리 질문 -> 와이파이/콘센트 질문 -> 확인
- 너무 많은 조건을 말하지 않고, 실용적인 질문 3개만 묻는다.
- 마지막에는 방문 전에 확인하고 싶다고 정리한다.

## 한국어 전문

안녕하세요. 오늘 오후에 카페에 가려고 하는데 몇 가지 질문이 있습니다.

저는 노트북으로 짧게 작업할 예정이라 조용한 자리가 있는지 알고 싶습니다.

혹시 와이파이가 안정적으로 잘 되는 편인가요?

그리고 콘센트를 사용할 수 있는 좌석도 있나요?

제가 보통 퇴근 후에 한 시간 정도 공부하거나 일을 해서, 너무 시끄럽지 않은 자리가 필요합니다.

만약 창가 자리나 벽 쪽 자리가 가능하면 더 좋을 것 같습니다.

예약이 필요한지도 궁금합니다.

방문 전에 미리 확인할 수 있으면 도움이 될 것 같습니다.

감사합니다.

## 영어 전문

Hello. I am planning to visit your cafe this afternoon, and I have a few questions.

I will do some short work on my laptop, so I would like to know if you have quiet seats.

Is the Wi-Fi usually stable there?

Also, are there any seats where I can use an outlet?

I usually study or work for about an hour after work, so I need a place that is not too noisy.

If a window seat or a seat near the wall is available, that would be even better.

Do I need to make a reservation?

It would help if I could check these things before I visit.

Thank you.

## 키 문장 및 단어

### Key Sentences

- I have a few questions.
- Do you have quiet seats?
- Is the Wi-Fi stable?
- Can I use an outlet?
- Do I need to make a reservation?

### Key Words

- quiet seat : 조용한 자리
- stable Wi-Fi : 안정적인 와이파이
- outlet : 콘센트
- reservation : 예약

## 키 한국어 문장 및 대응되는 영어표현 여러개

### 조용한 자리가 있는지 알고 싶습니다.

- I would like to know if you have quiet seats.
- Do you have any quiet seats available?
- Is there a quieter area in the cafe?

### 와이파이가 안정적인가요?

- Is the Wi-Fi stable?
- Does the Wi-Fi usually work well?
- Can I use reliable Wi-Fi there?

### 콘센트를 사용할 수 있는 좌석이 있나요?

- Are there any seats where I can use an outlet?
- Do you have seats with outlets?
- Can I charge my laptop at the cafe?

### 예약이 필요한가요?

- Do I need to make a reservation?
- Should I book a seat in advance?
- Is a reservation required?
