# Roleplay - Ask Questions

## 문제

- 한국어: 예약할 때 직원에게 몇 가지 질문을 해 주세요.
- English: You are making a reservation. Ask the staff a few questions.

## 암기 포인트 및 흐름

- 인사 → 시간 확인 → 인원 확인 → 자리/주차 확인 → 재확인 순서
- 짧고 공손하게 한 번씩만 묻는다
- 마지막에 꼭 확인하고 싶다는 말을 넣는다

## 한국어 전문

안녕하세요. 예약 관련해서 몇 가지 질문이 있습니다.
저희는 두 명인데, 오늘 저녁 7시쯤 자리가 가능한가요?
주차도 가능한지 같이 확인하고 싶습니다.
그리고 가능하면 창가 쪽처럼 조금 조용한 자리가 있으면 좋겠습니다.
아이가 있어서 너무 시끄럽지 않으면 더 좋겠습니다.
지난번에는 예약만 하고 세부를 못 물어봐서 조금 불편했는데, 이번에는 미리 확인하고 싶습니다.
마지막으로, 도착 전에 한 번 더 확인할 방법이 있는지도 알고 싶습니다.
답변해 주시면 감사합니다.

## 영어 전문

Hello. I have a few questions about the reservation.
There are two of us. Is a table available around 7 p.m. today?
I would also like to check whether parking is available.
And if possible, we would prefer a quiet window seat.
We are coming with a child, so a less noisy table would be helpful.
Last time, I only made the reservation and did not ask about the details, so it was a little inconvenient.
I would like to confirm everything before we arrive.
Thank you for your help.

## 키 문장 및 단어

### Key Sentences

- Is a table available around 7 p.m.?
- Is parking available?
- We would prefer a quiet window seat.

### Key Words

- reservation : 예약
- available : 가능한
- parking : 주차
- window seat : 창가 자리

## 키 한국어 문장 및 대응되는 영어표현 여러개

### 저녁 7시쯤 자리가 가능한가요?

- Is a table available around 7 p.m.?
- Can we book a table around 7 p.m.?
- Is 7 p.m. still open?

### 아이와 함께 가서 너무 시끄럽지 않으면 좋겠습니다.

- We are coming with a child, so a less noisy table would be helpful.
- A quiet table would be better because we are bringing our child.
- We would appreciate a calmer seat for our family.

### 주차가 가능한지 확인하고 싶습니다.

- I would like to check whether parking is available.
- Could you tell me if parking is available?
- Do you have parking there?

### 도착 전에 한 번 더 확인하고 싶습니다.

- I would like to confirm everything before we arrive.
- It would help to double-check before we go there.
- I want to make sure the details are clear before we come.

### 조금 조용한 자리가 있으면 좋겠습니다.

- We would prefer a quiet window seat.
- A quieter table would be great.
- If possible, we would like a less noisy seat.

## Follow-up Hooks

- Could you suggest another time?
- Is there a quieter area?
- Can we confirm by phone later?
- Is there parking nearby?
