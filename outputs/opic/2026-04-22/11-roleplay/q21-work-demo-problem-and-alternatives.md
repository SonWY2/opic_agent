# Roleplay - Work Demo Problem and Alternatives

## 문제

- 한국어: 데모 환경에 문제가 생겼습니다. 고객에게 상황을 설명하고 대안을 제시해 주세요.
- English: There is a problem with the demo environment. Explain the situation to the client and suggest alternatives.

## 암기 포인트 및 흐름

- 문제 설명 -> 사과 -> 영향 범위 -> 대안 2개 -> 확인 요청
- IT 업무 상황에서 자주 쓸 수 있는 안정적인 문제 해결 흐름이다.
- "완전히 망했다"가 아니라 "일부 기능이 불안정하다"로 말한다.

## 한국어 전문

안녕하세요. 오늘 데모와 관련해서 미리 말씀드릴 문제가 있습니다.

현재 데모 서버의 일부 기능이 조금 불안정합니다.

죄송하지만, 실시간 데모를 그대로 진행하면 중간에 화면이 느려질 수 있습니다.

대신 첫 번째 방법으로, 제가 미리 녹화해 둔 짧은 데모 영상을 보여 드릴 수 있습니다.

두 번째 방법으로, 핵심 기능만 먼저 설명하고 전체 데모는 내일 다시 진행할 수도 있습니다.

데이터 이전 흐름과 주요 화면은 오늘 설명드릴 수 있습니다.

다만 실제 클릭 테스트는 안정적인 환경에서 보여 드리는 것이 더 좋을 것 같습니다.

어떤 방식이 더 편하신지 알려 주시면 그에 맞춰 진행하겠습니다.

불편을 드려 죄송합니다.

## 영어 전문

Hello. I want to let you know about an issue before today's demo.

Some features on the demo server are a little unstable right now.

I am sorry, but if we run the live demo as planned, some screens may become slow during the meeting.

As a first option, I can show you a short demo video that I recorded in advance.

As a second option, I can explain only the key features today and run the full demo again tomorrow.

I can still explain the data migration flow and the main screens today.

However, it would be better to show the actual click test in a stable environment.

Please let me know which option is more convenient for you, and I will follow that.

I am sorry for the inconvenience.

## 키 문장 및 단어

### Key Sentences

- Some features are a little unstable.
- Some screens may become slow.
- I can show a recorded demo video.
- We can run the full demo again tomorrow.
- Please let me know which option is more convenient.

### Key Words

- unstable : 불안정한
- recorded demo : 녹화된 데모
- key features : 핵심 기능
- inconvenience : 불편

## 키 한국어 문장 및 대응되는 영어표현 여러개

### 일부 기능이 조금 불안정합니다.

- Some features are a little unstable.
- A few functions are not fully stable right now.
- The demo environment has some minor issues.

### 실시간 데모 중 화면이 느려질 수 있습니다.

- Some screens may become slow during the live demo.
- The system may respond slowly during the demo.
- There may be some delay if we run the live demo.

### 녹화해 둔 데모 영상을 보여 드릴 수 있습니다.

- I can show you a recorded demo video.
- I can use a demo video I prepared in advance.
- I can walk you through a pre-recorded demo.

### 전체 데모는 내일 다시 진행할 수 있습니다.

- We can run the full demo again tomorrow.
- I can reschedule the complete demo for tomorrow.
- We can do the full live demo tomorrow instead.
