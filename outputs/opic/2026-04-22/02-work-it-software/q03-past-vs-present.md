# Work / IT Software - Past vs Present

## 암기 포인트 및 흐름

- 예전에는 수동, 지금은 자동화와 검토 도구
- 빠른 편의성과 여전한 사람 판단을 같이 말한다
- 변화의 결과를 작은 업무 예시로 닫는다

## 한국어 전문

예전에는 반복 작업이 많아서, 같은 일을 수동으로 처리하는 시간이 길었습니다.
지금은 자동화 도구를 조금씩 써서, 기본 작업은 더 빠르게 넘길 수 있습니다.
예전에는 문제를 하나씩 직접 찾았지만, 지금은 로그와 도구를 먼저 봅니다.
특히 배포 전 확인 작업은 예전보다 훨씬 체계적으로 바뀌었습니다.
지난달에는 자동 확인 결과가 한번 애매하게 나와서, 결국 제가 직접 비교하면서 15분 더 봐야 했습니다.
물론 도구를 써도 마지막 판단은 사람이 해야 해서, 완전히 편해진 것은 아닙니다.
그래도 예전보다 실수가 줄어서, 일의 흐름은 확실히 좋아졌다고 느낍니다.
그래서 지금은 손이 덜 가는 대신, 판단은 더 정확해졌다고 말할 수 있습니다.

## 영어 전문

In the past, I had to do many repetitive tasks by hand.
Now I use automation tools little by little, so I can finish basic work faster.
In the past, I found problems one by one, but now I check logs and tools first.
Release checks are much more structured now than they used to be.
Last month, one automatic check gave a vague result, so I still had to compare things myself for about 15 more minutes.
Even with tools, the final decision still has to be made by a person, so it is not completely easy.
Still, mistakes happen less often now, so I feel the workflow has definitely improved.
So compared to the past, there is less manual work, but the judgment is more accurate.

## 키 문장 및 단어

### Key Sentences

- I use automation tools more now than before.
- Release checks are more structured now.
- I still need to make the final decision myself.

### Key Words

- structured : 체계적인
- automation : 자동화
- judgment : 판단

## 키 한국어 문장 및 대응되는 영어표현 여러개

### 예전에는 수동 작업이 많았습니다.

- I had to do many tasks by hand in the past.
- I used to do a lot of manual work.

### 지금은 도구를 써서 기본 작업을 더 빨리 처리합니다.

- I use tools now to handle basic work faster.
- I can finish simple tasks more quickly with tools.
- I rely on automation for the routine parts.

### 중요한 판단은 여전히 사람이 해야 합니다.

- People still need to make the important decisions.
- Human judgment is still necessary for important work.
- Tools help, but people still make the final call.

### 그래서 지금은 속도와 판단을 같이 챙깁니다.

- So now I try to balance speed and judgment.
- I focus on both speed and careful decisions.
- The goal is faster work without losing good judgment.

## Follow-up Hooks

- What automation tools do you use?
- What was harder in the past?
- Do you still do manual checks?
- What still needs human judgment?
- Has the workflow become more stable?
