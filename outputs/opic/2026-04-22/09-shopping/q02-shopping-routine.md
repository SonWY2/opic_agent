# Shopping - Routine

## 문제

- 한국어: 쇼핑할 때 보통 어떤 순서로 하는지 말해 주세요.
- English: Tell me about your usual shopping routine.

## 암기 포인트 및 흐름

- 필요 확인 -> 검색 -> 비교 -> 구매 -> 배송 확인
- 너무 자세한 기능 설명보다 쉬운 순서 표현을 반복한다.
- 카페/퇴근 후 루틴과 연결해서 자연스럽게 말한다.

## 한국어 전문

저는 쇼핑을 할 때 먼저 정말 필요한 물건인지 확인합니다.

충동적으로 사면 집에 물건이 너무 많아져서, 보통 장바구니에 먼저 넣어 둡니다.

그다음 퇴근 후나 주말 아침에 가격과 후기를 천천히 비교합니다.

특히 전자기기 액세서리는 호환이 되는지 꼭 확인합니다.

예를 들어 케이블이나 충전기를 살 때는 제 노트북과 맞는지 먼저 봅니다.

마음에 드는 물건이 있으면 바로 사기보다 배송 날짜와 반품 조건도 확인합니다.

마지막으로 주문한 뒤에는 배송 알림을 한 번씩 확인합니다.

이렇게 하면 시간이 조금 걸리지만, 실수를 줄일 수 있어서 좋습니다.

## 영어 전문

When I shop, I first check if I really need the item.

If I buy things too quickly, my home gets too crowded, so I usually put items in the cart first.

Then after work or on a weekend morning, I slowly compare prices and reviews.

For electronic accessories, I always check compatibility.

For example, when I buy a cable or charger, I first check if it works with my laptop.

When I find something I like, I also check the delivery date and return policy before buying it.

After I place the order, I check the delivery notification from time to time.

This takes a little time, but it helps me avoid mistakes.

## 키 문장 및 단어

### Key Sentences

- I first check if I really need the item.
- I put items in the cart first.
- I compare prices and reviews.
- I check the delivery date and return policy.
- It helps me avoid mistakes.

### Key Words

- cart : 장바구니
- compatibility : 호환성
- delivery date : 배송일
- return policy : 반품 조건

## 키 한국어 문장 및 대응되는 영어표현 여러개

### 먼저 정말 필요한 물건인지 확인합니다.

- I first check if I really need the item.
- I ask myself if the item is really necessary.
- I try to make sure I actually need it.

### 가격과 후기를 비교합니다.

- I compare prices and reviews.
- I check different prices and customer reviews.
- I look at reviews before I make a decision.

### 배송 날짜와 반품 조건도 확인합니다.

- I also check the delivery date and return policy.
- I make sure when it will arrive and how returns work.
- I check delivery and return details before buying.

### 실수를 줄일 수 있어서 좋습니다.

- It helps me avoid mistakes.
- This routine helps me make better choices.
- I can reduce mistakes when I check carefully.

## Follow-up Hooks

- Do you check reviews?
- How long do you compare products?
- What do you do after ordering?
- Have you ever returned something?
