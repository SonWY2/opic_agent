description: Build and save a full OPIc topic-folder pack
agent: opic-coach
---

Build a complete OPIc study pack from the user's selected topics and speaking level.

Default behavior:
- save results to `outputs/opic/YYYY-MM-DD/`
- create root companion files
- create one folder per topic
- create one markdown file per question inside each topic folder

Mandatory rule for every question file:
- include 암기 포인트 및 흐름
- include 한국어 전문
- include 영어 전문
- include 키 문장 및 단어
- include 키 한국어 문장 및 대응되는 영어표현 여러개

At the end, show only the saved folder/file list and any important notes.
