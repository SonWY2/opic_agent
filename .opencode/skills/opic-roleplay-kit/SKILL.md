## What it does
- Creates a linked Q11/Q12/Q13 flow
- Keeps the phone-call tone natural
- Ensures alternatives and follow-up logic are present

## Q11 pattern
Ask 3 to 4 practical questions:
- availability
- time
- location
- price / reservation / required item

## Q12 pattern
1. say the problem
2. apologize
3. give 2 alternatives
4. ask what works best

## Q13 pattern
1. introduce a similar real experience
2. explain the pain point
3. explain the fix
4. end with the result
