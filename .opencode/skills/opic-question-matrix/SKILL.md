- Reuses the same story asset across multiple question types
- Prevents missing coverage
- Suggests default topic folders and question files

## Question families
For each topic, try to cover:
1. describe
2. routine
3. compare past and present
4. memorable experience or problem
5. opinion or issue bridge

## Rule
Prefer 2 reusable stories per major domain, not one separate script per question.
