## What it does
- Keeps English in a safe spoken band
- Reduces over-complex wording
- Makes answers easier to recall

## Safe-language rules
- Prefer short and clean sentences
- Prefer concrete nouns over abstract ideas
- Use common verbs first
- Use one clear emotional reaction
- Use one clear time marker
- Use one clear mini episode

## Avoid
- stacked clauses
- academic tone
- long metaphors
- rare idioms
- overly formal transitions

## Quick rewrite pattern
Hard:
"I find it particularly meaningful because it provides me with psychological stability."

Safer:
"I like it because it helps me feel calm and stable."
