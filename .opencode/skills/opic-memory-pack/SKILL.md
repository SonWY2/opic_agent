## What it does
- Builds a memory flow map
- Builds lite scripts and cue cards
- Adds shadowing chunks
- Preserves meaning while reducing recall load

## Memory anchors
Keep these 6 anchors:
- opening
- detail 1
- detail 2
- comparison or episode
- feeling or lesson
- closing

## Recovery rule
If the speaker forgets details, the flow map and cue card must still allow a complete answer reconstruction.
